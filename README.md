# Urdu–Punjabi Shahmukhi NLP

This repository presents a reproducible pipeline for constructing a Punjabi Shahmukhi corpus via Gurmukhi transliteration and training word embeddings for low-resource NLP.

---

## 📁 What is included

* Urdu corpus
* Punjabi Shahmukhi corpus
* Trained Word2Vec models
* PCA and t-SNE visualizations
* Notebook for experiments
